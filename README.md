**Кафетерий льгот** — это веб-сервис, разработанный для автоматизации процесса выбора и запроса корпоративных льгот сотрудниками. Проект упрощает коммуникацию между персоналом и HR-отделом, предоставляя интуитивно понятную платформу для управления льготами и их приобретения.

## 🚀 Основные возможности

* **Для сотрудников:**
    * Просмотр каталога доступных льгот.
    * Выбор льгот из доступных на основании условий выдачи льготы.
    * Отслеживание статуса своих заявок.
* **Для HR-менеджеров:**
    * Управление списком льгот.
    * Рассмотрение и одобрение заявок от сотрудников.
    * Просмотр статистики по льготам.
